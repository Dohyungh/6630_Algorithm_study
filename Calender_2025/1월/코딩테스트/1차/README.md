# 코딩테스트

일자 : 2025.1.9(목)
시간 : 10:00 ~ 12:00

## 제출 방법

1. 같이 올라온 3개의 문제를 읽고 IDE에서 문제를 푼다.
2. BOJ에 제출하여 자체적으로 채점한다.
3. 종료시간 전에 문제 폴더에 `Main_이름.java`형식으로 작성해 자기 브랜치에 푸시한다.
4. 시험종료 후 `master` 브랜치에 머지한다.

## 유의점

1. 제출시에만 BOJ에 로그인한다.
2. **가능한 티어와 알고리즘 분류를 보지 않는다.**
