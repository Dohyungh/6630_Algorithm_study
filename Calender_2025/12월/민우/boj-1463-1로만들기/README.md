# ✔ 문제 풀이

- 주어진 정수 크기만큼 dp 배열을 생성한다.
- dp 배열을 순회하며 값을 채워주는데 6의 배수 인덱스면 3과 2 둘 다 나누어 떨어지니 3가지 경우에 대해 최솟값 + 1, 2 또는 3으로 나누어 떨어지면 1을 빼는 경우까지 2가지 경우에 대해 최솟값 + 1로 값을 찾아준다.
