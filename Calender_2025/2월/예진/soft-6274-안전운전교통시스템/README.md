### 설계 : 15분

- 각 교차로와 시간에 따른 신호 배열, 신호에 따른 진입방향과 전진 가능 방향에 대한 배열을 잘 정리한다.
- 처음 방문하는 교차로만 카운트 하며 dfs 한다.
- 이때 방문하는 교차로의 현재 신호의 진입방향과 교차로로 오던 방향이 일치해야 다음으로 넘어갈 수 있다.
- 신호는 4T마다 반복되기 때문에 신호가 반복되는 주기 안에 싸이클이 발생하면 중단한다.
    - entered배열을 사용해서 특정 교차로에 특정 주기에 특정 방향으로 진입한 적이 있는지 판단한다.
