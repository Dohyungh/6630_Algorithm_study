## 문제

임의의 자연수가 주어지면, 이를 네 개의 소수의 합으로 분해하는 프로그램을 작성하시오. 예를 들어 38 = 5 + 7 + 13 + 13이 된다.



## 입력
첫째 줄에 자연수 N(1 ≤ N ≤ 1,000,000)이 주어진다.



## 출력
첫째 줄에 네 개의 소수를 빈 칸을 사이에 두고 순서대로 출력한다. 불가능한 경우는 -1을 출력한다.

예제 입력 1   
```
38
```

예제 출력 1   
```
5 7 13 13
```

예제 출력 2
```
1 1 5 5
6 10 10 6
```

예제 출력 2    
```
0
```
